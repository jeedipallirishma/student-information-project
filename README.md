# Student Information Project

A beginner JavaScript project that collects student details and checks pass/fail status.

## Features
- Student name input
- College name input
- Section input
- Marks input
- Pass/Fail result generation

## Technologies Used
- HTML
- JavaScript

## Concepts Practiced
- Variables
- Data Types
- prompt()
- if/else statements
- document.write()
